# COBS
COBS-Consistent-Overhead-Byte-Stuffing

#### 算法原理
COBS算法通过以下方式工作：

##### 编码过程：
1. 每个块以一个计数字节开头，表示到下一个零字节的距离
2. 原始零字节被其到下一个零字节的距离值替代
3. 非零字节保持不变

##### 解码过程：
1. 读取计数字节，确定要复制的非零字节数
2. 在适当位置恢复零字节

#### 通信场景用 COBS：专门消除 0x00，工业标准

1、Base64 是通用最快方案，输出天然无 0x00

2、COBS 是通信领域专门消除 0x00 的标准算法

|算法        |    难度 | 膨胀率  |  适用场景|
|-------|-------|-------|-------|
|Base64      |    极低 | +33%   |  通用、快速开发、跨语言|
|COBS        |    低   | +1~5%  |  串口 / 通信、专业场景|
|8→9 位编码	  |    中   | +12.5% |  自己实现、追求简单稳定|
|128 进制映射 |    高  | 最低    | 极致空间优化|
